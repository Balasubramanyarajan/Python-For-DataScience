---

## 📂 Repository Structure

The repository is organized by topic-specific folders, each containing hands-on Jupyter Notebooks (`.ipynb`):

* **`Numpy/`** – Multi-dimensional array operations, vectorization, mathematical functions, and indexing techniques.
* **`Pandas/`** – Data cleaning, Series and DataFrame manipulations, handling missing values, grouping, and merging datasets.
* **`Matplotlib/`** – Core data visualization techniques, line plots, scatter plots, bar charts, and figure customizations.
* **`Seaborn/`** – Advanced, statistically-driven statistical visualizations, heatmaps, pairplots, and distribution plots.

## 🚀 Getting Started Locally

If you want to clone this repository and run these notebooks on your machine:

1. **Clone the repository:**
   git clone [https://github.com/Balasubramanyarajan/Python-For-DataScience.git]
   cd Python-For-DataScience

2. **Environment setup:**
    Open your terminal or Git Bash and run:
        pip install numpy pandas matplotlib seaborn jupyterlab

3. **Launch Jupyter Lab:**
    Open your terminal or Git Bash and run:
        Jupyter Lab
